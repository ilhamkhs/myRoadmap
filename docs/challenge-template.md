# Template Challenge Sederhana

Pakai format ini kalau mau bikin latihan baru.

## 1) Judul

Contoh: `Challenge: Cek Angka Genap`

## 2) Tujuan

Contoh: `Latihan if/else sederhana`

## 3) Soal

Tulis soal dalam 1-3 kalimat, singkat dan jelas.

Contoh:
`Buat fungsi yang menerima 1 angka.`
`Jika angkanya genap, tampilkan "Genap".`
`Jika tidak, tampilkan "Ganjil".`

## 4) Contoh Input

Contoh:
`4`

## 5) Contoh Output

Contoh:
`Genap`

## 6) Petunjuk (boleh dikosongkan)

Contoh:
`Gunakan operator % untuk cek sisa bagi.`

## 7) Selesai Jika

- Kode bisa dijalankan.
- Hasil sesuai yang diminta soal.
- Kodenya rapi dan mudah dibaca.
