# JavaScript Beginner Roadmap

Bagian ini berisi materi JavaScript dasar untuk pemula.
Tujuannya: belajar pelan-pelan dari konsep paling awal sampai latihan logika sederhana.

## Cara Belajar

- Ikuti urutan folder dari angka kecil ke besar.
- Baca komentar di awal file, lalu jalankan kodenya.
- Ubah sedikit contoh nilainya supaya kamu lihat perbedaan hasil.
- Kalau ada folder `challenge`, coba kerjakan dulu sebelum lihat jawaban.
- Tidak perlu buru-buru. Fokus paham konsepnya satu per satu.

## Urutan Materi

1. `1-dataType`  
   Belajar variabel, teks, angka, dan boolean.

2. `2-conditional`  
   Belajar `if`, `else`, dan `else if` untuk ambil keputusan.

3. `3-object`  
   Belajar menyimpan data dalam bentuk object.

4. `4-array`  
   Belajar array dasar sampai `forEach`.

5. `5-function`  
   Belajar fungsi agar kode bisa dipakai berulang.

6. `6-arrowFunction`  
   Belajar versi singkat dari penulisan fungsi.

7. `7-scopeAndReturn`  
   Belajar nilai balik fungsi (`return`) dan batas variabel.

8. `8-loop`  
   Belajar perulangan `for`, `while`, `for...of`, dan `for...in`.

9. `9-Sorting`  
   Belajar penyusunan data sederhana (bubble sort).

10. `10-Searching`  
    Belajar mencari data dengan linear search dan binary search.

11. `11-CountData`  
    Belajar menghitung jumlah data dengan beberapa pola.

12. `12-CountMinMax`  
    Belajar mencari nilai paling besar dan paling kecil.

13. `13-callback`  
    Belajar callback dan alur proses bertahap.

## Saran Pola Latihan

- Hari 1-2: `1-dataType` sampai `3-object`
- Hari 3-4: `4-array` sampai `6-arrowFunction`
- Hari 5-6: `7-scopeAndReturn` sampai `8-loop`
- Hari 7+: `9-Sorting` sampai `13-callback`

## Target Hasil Belajar

Setelah selesai `1Beginner`, kamu diharapkan bisa:

- Menjelaskan beda `let` dan `const` dengan kata sendiri.
- Membuat alur keputusan sederhana dengan `if/else`.
- Mengolah data array sederhana dengan loop.
- Membuat fungsi kecil untuk hitung atau cek data.
- Membaca kode JavaScript dasar dengan lebih percaya diri.

```bash

```

Ganti path file sesuai materi yang ingin kamu coba.
