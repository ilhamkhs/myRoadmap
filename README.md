# My JavaScript Roadmap

Roadmap ini dibuat untuk membantu belajar JavaScript dari dasar sampai lanjut.
Fokusnya adalah contoh sederhana, bahasa yang mudah, dan latihan yang bertahap.

## Struktur Belajar

- `1Beginner` → materi dasar untuk pemula.
- `2Advance` → materi lanjutan setelah paham dasar.

## Cara Menggunakan Roadmap

1. Mulai dari `1Beginner` sesuai urutan angka folder.
2. Jalankan file satu per satu, lalu ubah sedikit nilainya.
3. Kerjakan file challenge tanpa melihat jawaban dulu.
4. Setelah nyaman, lanjut ke `2Advance`.

## Learning Path Sederhana

- Minggu 1: fokus `1Beginner` bagian data type, conditional, object, array.
- Minggu 2: fokus function, arrow function, return, loop.
- Minggu 3: fokus sorting, searching, callback, dan challenge.
- Minggu 4: mulai `2Advance` sambil buat mini project.

## Menjalankan Contoh Kode

```bash
node "1Beginner/1-dataType/1-variable.js"
```

Ganti path file sesuai materi yang ingin kamu pelajari.
