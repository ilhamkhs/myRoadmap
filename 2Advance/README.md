# JavaScript Advance Roadmap

Bagian ini untuk learner yang sudah nyaman dengan materi dasar di `1Beginner`.

## Tujuan

- Memahami cara kerja JavaScript lebih dalam.
- Menulis kode yang lebih terstruktur.
- Mulai belajar pola yang dipakai di project nyata.

## Urutan Belajar Disarankan

1. `OOP`  
   Belajar class, method, dan enkapsulasi sederhana.

2. `this-and-context`  
   Belajar perilaku `this` pada function dan object.

3. `asynchronous`  
   Belajar callback, Promise, dan `async/await`.

4. `error-handling`  
   Belajar `try/catch` dan membuat pesan error yang jelas.

5. `modules`  
   Belajar memecah kode ke banyak file agar rapi.

## Cara Pakai

- Ikuti folder sesuai urutan di atas.
- Jalankan setiap file dan baca outputnya.
- Ubah contoh nilai agar paham perubahan hasil.
- Selesaikan challenge jika tersedia.
